Navigation

The navigation application illustrates the use of a standard image-guided navigation system. Note that both the reference marker and the pointer tool's marker must be detected by the tracking systems at all times. This is indicated by the corresponding LED colors, green detected, red not detected.

To perform an "intervention" you will need to follow the three step process described below:

  1. Preparation:

    If you haven't done so:

    1. Perform camera calibration using the provided application.

    2. Perform pivot calibration using the provided application.

    3. Generate the navigation setup using the provided planning application.

  2. Initialization:

    1. Press the "Load Configuration" button (loads the file IGITutorial/Configuration/IGIConfiguration.xml).

    2. Check that the camera can see the area where you intend to navigate (popup window will close after a couple of seconds).

    3. You will now identify corresponding fiducial markers in the image and the physical world. Select a fiducial from the fiducials tab. Touch the corresponding physical fiducial and press the "Accept Fiducial" button while the pointer tip is still touching the fiducial. Repeat the process for all fiducials.

    4. Press the "Register" button.

  3. Navigate:

    1. Move the pointer tool and the display is updated based on the position of the tool's tip. If you select a target from the targets tab the distance between the current pointer tool tip and the defined target position in image space will be displayed.
