Remote Control

This program allows you to control a power point presentation using your computer's webcam and a set of markers. It is a toy example showing how a tracking device can be used as part of a computer human interface.

  1. Preparation:

    If you haven't done so:

    1. Perform camera calibration using the provided application.

    2. Print the tracked marker patterns (IGITutorial/Data/TrackedMarkers/markers.pdf), and separate the markers along the dashed lines. Make sure that the internal, black, marker square size is 50mm.

    3. Cut them out and attach to a rigid planar surface. A standard index card is sufficiently rigid for our purposes.

  2. Use the remote:

    1. Check that the camera can see the area where you intend to work, "Check Camera View". Popup window will close after a couple of seconds.

    2. When the program is running and a marker is visible to the camera it will:

      • marker 202: send a "Page Down" keystroke to the application that has focus.
      • marker 502: send a "Page Up" keystroke to the application that has focus.
      • marker 201: terminate the remote control program.

Remember to switch focus to the program you want to control. If the focus is on the "Remote Control" application nothing will happen.
